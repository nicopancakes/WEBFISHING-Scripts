# WEBFISHING-Scripts
Using GDScript with Finapse X to Mod WEBFISHING.

# How to Run:

1. **Install R2Modman - https://r2modman.com/download-latest/**
---
2. **Install FinapseX Under Mods**
---
3. **Join a Server & Press F1**
---
4. **Paste the Script into The Executor and Press 'Run'**

---
## Scripts:
---
1. [moneyadd.gd](https://github.com/nicopancakes/WEBFISHING-Scripts/blob/main/scripts/moneyadd.gd) - Add $1,000,000 when Executed.
2. [moneyremove.gd](https://github.com/nicopancakes/WEBFISHING-Scripts/blob/main/scripts/moneyremove.gd) - Removes all Money (moves to $0)
